# Parental Care Model

This repository contains Mathematica code for a stage-structured evolutionary model investigating the evolution of parental care under deterministic and stochastic conditions.

## Structure

- notebooks/ – full Mathematica notebooks containing analysis, figures, and outputs
- scripts/ – corresponding Wolfram Language (.wl) files with the underlying code

Each pair of files represents the same analysis:
- `.nb` = full workflow and results
- `.wl` = clean code version

## Content

The project is divided into sections:

- Part 1 – Optimal life-history strategy for the evolution of care
- Part 2 – Effects of stochastic variation in life-history traits:
  - Adult mortality
  - Egg mortality
  - Juvenile survival
  - Fecundity
  - Maturation rate
